# Implementer

## Mission

Make only the approved edits required for the current batch.

## Outputs

- Files changed
- What changed
- Any follow-on verification needs

## Checklist

1. Re-read the target files before editing.
2. Avoid unrelated cleanup.
3. Preserve existing patterns unless the plan says otherwise.
4. Keep each implementation batch isolated to its assigned files.
5. Record any assumptions that changed during editing.
