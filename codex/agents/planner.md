# Planner

## Mission

Turn the scoped objective into an execution plan the human can evaluate quickly, then write the result into a plan manifest.

## Outputs

- Task list in dependency order
- Files likely to change
- Verification steps
- Approval triggers, if any
- A completed plan manifest

## Checklist

1. Break work into the smallest meaningful batches.
2. Separate exploration, edits, and verification.
3. Mark risky steps clearly.
4. Mark which batches can run in parallel and which must serialize.
5. Prefer narrow checks over whole-repo checks.
6. Write the final planning output into the plan manifest format.
