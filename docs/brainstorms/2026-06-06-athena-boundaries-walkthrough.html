What to remember

The same boundary rule repeats.

Let each layer own one decision, pass forward the minimum safe evidence, and refuse authority that belongs somewhere deeper.

Route access is not command authority. A mounted shell can orient the operator, but sale-affecting work still needs terminal, staff, drawer, and command gates.
Local-first means durable local records. Checkout work records locally first, then sync projects facts into cloud records when validation is available.
Cloud uncertainty is not always a blocker. When local recording remains safe, field sales can continue and review happens behind the scenes.
Evidence is not raw internals. Operational surfaces should show actors, quantities, statuses, and next steps without exposing secrets or raw backend text.
