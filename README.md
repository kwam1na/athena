# Athena: Inventory Management App

Athena is a simple yet powerful inventory management app for businesses. Built with Next.js, TypeScript, Supabase, and Tailwind CSS, Athena aims to provide an efficient and user-friendly experience for managing inventory.

## Features

- Easily manage inventory items and their details.
- Organize items in categories and subcategories for quick access.
- Track inventory transactions with timestamps and user info.
- Support for multiple stores or businesses with separate inventories and users.
- Secure user authentication and authorization.

Athena combines modern web technologies and best practices to create a scalable and maintainable solution that caters to businesses' inventory management needs.
